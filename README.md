# SEKU Send Money Flutter App

This is a minimal but complete Flutter app demonstrating:
- Login (mock)
- Dashboard
- Send Money page with validation
- Reusable SendButton widget
- Animated success message
- Fade transitions between pages

## How to run

1. Create a new Flutter project (or empty folder) and copy the files above into the project structure:
   - Replace `pubspec.yaml`
   - Create `lib/` and the subfolders/screens as shown.
2. Run:
flutter pub get
flutter run
